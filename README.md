# Aria's Code 🌸

A magical, premium toddler app built with React — designed for baby girls with a beautiful baby pink and baby blue theme.

## Features

### 🎹 Play Piano
- 15 white keys + 10 black keys with correct musical frequencies
- Web Audio API synthesized sounds with ADSR envelopes
- Colorful confetti bursts and floating music notes on every key press
- Rainbow-colored keyboard layout

### 🫧 Pop Bubbles!
- Glossy, realistic bubbles float up from the bottom of the screen
- Tap to pop with premium 40-particle firework explosions and shockwave rings
- Satisfying pop sounds generated with Web Audio API
- Score counter with animated bounce

### 🔢 Learn Numbers (1–20)
- Large, animated numbers with glowing effects
- Speech synthesis reads each number aloud in a child-friendly voice
- Celebration confetti on every tap
- Trophy celebration screen after completing all 20 numbers
- Progress bar to track learning journey

## Design

- Animated gradient background shifting between baby pink, lavender, and baby blue
- 35+ animated sparkle particles across all screens
- Glossy shimmer overlay for a premium feel
- Google Fonts: **Fredoka One** (headings) and **Nunito** (body)
- Fully responsive — optimized for mobile (375–430px) and desktop

## Tech Stack

- **React 18** (via CDN)
- **Tailwind CSS** (via CDN)
- **Web Audio API** — all sounds synthesized programmatically (no external audio files)
- **Web Speech API** — number voiceovers
- Single HTML file, zero build step

## Getting Started

Just open `index.html` in any modern browser. No installation needed.

```bash
# Or serve locally
npx serve .
```

## Live Demo

Open `index.html` directly in your browser — everything runs client-side.
